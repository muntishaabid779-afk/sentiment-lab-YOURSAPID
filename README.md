
# Legal Notice Classification

## Description
This project classifies legal notices into:
- Contract Dispute
- Intellectual Property Claim
- Regulatory Compliance

## Models
- Logistic Regression
- Multinomial Naive Bayes

## Installation

pip install -r requirements.txt

## Configuration
All parameters are stored in config.json

## Results
The best model was selected based on F1-score and accuracy.
## Model Evaluation
Models were evaluated using accuracy and F1-score.
